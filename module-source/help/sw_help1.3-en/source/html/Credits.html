Sower Applications are developed with CodeBlocks, and gcc. Using wxWidgets for user interface elements, MiniAudio for audio, and libbz2 for file compression.

All of Sower Applications has been designed to work with ThML files as defined in ThML 1.04.htm which can be found at 'https://www.ccel.org/ThML/ThML1.04.htm' . The reference information in thml.smd comes from the same file. To learn more go to ccel.org .

The jpeg library is used for displaying jpeg images, and this notice is required:

'This software is based in part on the work of the Independent JPEG Group.'
