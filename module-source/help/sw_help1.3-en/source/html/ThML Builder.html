ThML Builder is a utility designed to create a ThML file from multiple html files. The resulting file may require further editing. It is intended to reduce some of the work required to convert a multi-file html book to ThML.

To create a ThML file:

  • Enter any information that should be stored in the dublin core of the file.
  • Click the 'Select files' button to select the files to be used in the build.
  • Select or enter the 'Save as' file path.
  • Uncheck any files in the list that should not be included in the build.
  • Use the 'Up', and 'Down' buttons to position the files according to the desired contents of the resulting ThML file.
  • Click 'Build'.

ThML Builder can also load a Sower Project file which describes how to build the ThML file. This method allows for the creation of more complex books. And also provides options for automatic markup of scripture references.
