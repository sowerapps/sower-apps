SowerBase is a cross-platform dynamic link library, or shared object designed to provide the foundation code for Sower applications. Some of it's capabilities are:

  • Handles theme, translation, bookmark, and preferences files for applications.
  • Provides facilities for working with Html, and ThML.
  • Recognizes scripture references in multiple languages, and forms.
  • GeoLocation data for working with maps of the ancient world.
  • Provides a very versatile book module design, and the facilities to work with those modules.
  • Provides a gui frame work on top of the wxWidgets frame work, reducing much of the work involved in handling windows, menus, and toolbars. Including specialized controls designed to work with Sower modules, ThML files, and automatically update themselves for language, or theme changes.
  • Plug-In, and module managers.
  • Initialization of applications, and SowerBase itself.

And more. Greatly reducing the complexity of writing document driven software that works with ThML files, or Sower modules.
