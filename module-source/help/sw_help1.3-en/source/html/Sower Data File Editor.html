Sower Data File Editor is used to edit Sower's table based data files (sdf), and is similar to Sower Theme Editor, and Sower Translation Editor. It is designed for general use and allows more direct editing of the data in those files, and does not require that each table have the same list of identifiers.

To edit a file first open it with the 'Open' command on the 'File' menu.

Table Editor will show a list of the  tables available in the file.

Table Panel:

Select a table, and click 'Load' to open that table for editing. Click 'Save' to save changes to the table before clicking 'Close' which will close the loaded table without saving any changes.

To add a new table enter a name in the box provided, and click 'Add'. Click 'Load' to open the new table for editing.

Clicking 'Delete' will delete any selected table.

Values are edited by clicking the item in the list at the of the application frame.

Identifiers Panel:
String identifiers can only be added, or deleted when a table is loaded. And are only added to the loaded table.

Enter an identifier and click 'Add' after loading a table to add a new identifier.

Clicking 'Delete' will delete any selected identifier from the loaded table.

Values are edited by clicking the item in the list at the of the top of the application frame.
