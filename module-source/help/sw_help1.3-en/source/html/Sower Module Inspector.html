Sower Module Inspector is a utility to examine modules created with Sower Module Writer.

  1. The module header.
  2. The table of contents.
  3. The media contents, external files referenced in the source file that have been automatically included in the module.
  4. Scripture elements that have been identified in the source file. These are not presented in a list, to verify this enter a reference in the box provided and click the 'Find' button such as 'Gen. 1:1'.
  5. ScripCom elements that have been identified in the source file. Check for entries as with Scripture elements.
  6. Strongs Greek entries.
  7. Strongs Hebrew entries.
  8. Terms (entries) in dictionaries, or encyclopedias.
  9. Notes, a complete list of notes identified in the source file.

For encrypted modules there must be a master key, or a user key available to open the module.
